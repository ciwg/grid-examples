Review

Start here, open one queue, inspect one record, and only then branch into authoring or operation work.

Review stays the browser default because it is the main job: find the next record that needs attention and act from that record.

Primary Flow

  1. Review the queue that best matches what you know: draft work, repeated problems, or a known record.
  2. Open one current record and stay with it until the next action is clear.
  3. Act from that record instead of re-entering context in blank forms.
